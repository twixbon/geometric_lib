- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

# Area functions
## Circle
- Принимает число r, возвращает квадрат r, умноженный на число pi 
    - Пример:
        - Ввод:
        - r = 3
        - Вывод:
        - area = 28.26

## Square
- На вход принимает число a, возвращает квадрат числа a
    - Пример:
        - Ввод:
        - a = 2
        - Вывод:
        - area = 4

## Rectangle
- Принимает числа a и b, возвращает произведение a на b
    - Пример:
        - Ввод:
        - a = 5
        - b = 4
        - Вывод:
        - area = 20

## Triangle
- Принимает числа a и h, возвращает произведение a и h, делённое на 2
    - Пример:
        - Ввод:
        - a = 4
        - h = 2
        - Вывод:
        - area = 4

# Perimeter functions
## Circle
- Принимает число, возвращает удвоенное произведение r на число pi
    - Пример:
        - Ввод:
        - r = 3
        - Вывод:
        - perimeter = 18.84

## Square
- Принимает число a, возвращает произведение a * 4
    - Пример:
        - Ввод:
        - r = 2
        - Вывод:
        - perimeter = 8

## Rectangle
- Принимает a и b, возвращает сумму a и b, умноженную на 2
    - Пример:
        - Ввод:
        - a = 3
        - b = 4
        - Вывод:
        - perimeter = 14

## Triangle
- Принимает числа a, b, c, возвращает сумму a, b и c
    - Пример:
        - Ввод:
        - a = 1
        - b = 2
        - c = 3
        - Вывод:
        - perimeter = 6
